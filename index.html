  • O javascript foi lançado inicialmente no final de 1995.

  • Em 2005 houve a ascensão do Ajax e JQuery

  • E então 2010 houve o BOOM dos frameworks de JS.
    • Periodo que vivemos até hoje
    • Algumas como React; Vue.js; Angular

Operadores

  • Aritméticos
    • Adição (+) e Substração (-)
    • Multiplicação (*) e Divisão (/)
    • Módulo (%)
    • Incremento (++) e Decremento (--)
  • Comparação
    • Menor que (<) e Maior que (>)
    • Menor igual (<=) e Maior igual (>=)
    • Igual (==) e Não igual (!=)
    • Igual 2 (===) (?) e Não igual 2 (!==)
  • Lógicos
    • E (&&)
    • OU (||)
    • NÃO (!)

    • Atenção

Funções

  • Entender funções em javascript é fundamental!!!
  • Uma função é um bloco de código que executa uma tarefa específica.
  • A princípio javascript possui funções de primeira classe
    • Basicamente diz que funções são como valores
    • Um tipo primitivo de dado

Objetos 📦

  • Além de entender funções é primordial entender o que são objetos no contexto do javascript.
  • Um objeto se assemelha com um dicionário em python
  • É um tipo de tabela hash
  • Um objeto em javascript é composto por chaves e valores
